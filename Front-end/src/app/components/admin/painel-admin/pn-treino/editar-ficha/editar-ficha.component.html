Guia de Montagem de Treino
1
Selecione o Exercício

No painel esquerdo (cinza), filtre pelo Grupo Muscular e escolha o Exercício desejado.

2
Defina a Carga de Trabalho

Preencha Séries e Repetições. Use o campo "Observações" para técnicas como Drop-set ou Isometria.

3
Adicionar à Lista

Clique no botão amarelo "Adicionar Exercício". O exercício irá para a tabela no lado direito (Lista de Exercícios).

4
Identifique a Ficha

No painel azul "Ficha", defina o Tipo (ex: A, B) e a Ordem (1, 2) para organizar a sequência no app.

Finalização:
Após adicionar todos os exercícios desejados à lista da direita, clique no botão verde grande "SALVAR FICHA COMPLETA" para gravar tudo no sistema de uma vez.
