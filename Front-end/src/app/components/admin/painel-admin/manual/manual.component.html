Guia de Uso Completo: Sistema Fitness Prime

Introdução

Bem-vindo ao Fitness Prime! Este manual foi criado para ser seu guia completo, ensinando não apenas o que cada botão faz, mas como utilizar o sistema de forma estratégica para otimizar a gestão da sua academia.

Menu Principal: Seu Ponto de Partida

O menu à esquerda é o coração da navegação. A partir dele, você acessa todas as ferramentas de gestão. Cada item leva a um módulo específico para gerenciar uma área da sua academia.

Como Gerenciar Seus Clientes

Manter um cadastro de clientes organizado é fundamental. Este módulo centraliza todas as informações e ações relacionadas aos seus alunos.

Passo a Passo para Cadastrar um Novo Cliente

1
No menu lateral, clique em Clientes.
2
Na tela de clientes, clique no botão azul Adicionar.
3
Preencha o formulário com as informações do aluno. Quanto mais completo, melhor será seu controle.
4
Clique em Salvar para confirmar.

Como Montar Fichas de Treino Personalizadas

A criação de fichas de treino é uma das funcionalidades mais importantes para a personalização e o acompanhamento dos alunos.

Passo a Passo para Criar uma Nova Ficha de Treino

1
Na lista de clientes, encontre o aluno e clique em Cadastrar Ficha (para alunos sem treino) ou Nova ficha (para adicionar treinos).
2
Adicione os Exercícios: Na seção "Adicionar Exercício", preencha os campos (Grupo, Exercício, Séries, Repetições, etc.) e clique no botão amarelo Adicionar Exercício. Repita para todos os exercícios do treino.
3
Organize a Ficha: Na seção "Ficha", defina o Tipo de Treino (A, B, C...) e a Ordem (1, 2, 3...).
4
Revise tudo e clique em Salvar. Para entregar ao aluno, use o botão Gerar PDF.

Como Gerenciar Sua Equipe

Este módulo permite cadastrar os membros da sua equipe e definir seus níveis de acesso ao sistema.

Passo a Passo para Adicionar um Novo Funcionário

1
No menu lateral, clique em Funcionários.
2
Clique no botão Adicionar.
3
Preencha os dados do funcionário.
4
Selecione a Função: Este é um passo crucial. A função (ex: Administrador, Personal) determina o que o usuário poderá ver e fazer no sistema. Escolha com atenção.
5
Clique em Salvar.
Para gerenciar: Na lista de funcionários, use o botão azul (Editar) para atualizar informações ou o vermelho (Excluir) para remover o acesso de um funcionário que não faz mais parte da equipe.

Como Configurar Seus Planos e Serviços

Aqui você define os produtos que sua academia vende, desde as mensalidades até serviços avulsos. Esta configuração é a base para o controle de pagamentos.

Gerenciando Suas Ofertas

A tela exibe seus Planos Principais e Extras (serviços adicionais).

  • Para criar uma nova opção: Clique em Adicionar. Você pode criar um "Plano Semestral" ou um serviço como "Avaliação Nutricional".
  • Para ajustar preços: Use o botão azul (Editar). É ideal para reajustes anuais de mensalidades ou alteração no valor de um serviço.
  • Para remover uma opção: Use o botão vermelho (Excluir) se um plano ou extra não for mais oferecido.

Como Gerenciar a Loja e Realizar Vendas

Este módulo funciona como um Ponto de Venda (PDV) e controle de inventário para os produtos da sua academia.

Passo a Passo para Realizar uma Venda

1
Na lista de produtos, encontre o item que o cliente deseja e clique no botão verde Adicionar.
2
O item aparecerá na seção "Itens na Venda". Se o cliente comprar mais de uma unidade, altere o campo "Qtd". O valor total será recalculado automaticamente.
3
Repita os passos 1 e 2 para outros produtos, se houver.
4
Confira os itens e o valor total. Quando estiver tudo certo, clique em Finalizar Venda.
Controle de Estoque Automático: Ao finalizar uma venda, o sistema automaticamente dá baixa na quantidade vendida do estoque, mantendo seu inventário sempre atualizado e evitando vendas de produtos que não estão disponíveis.

Como Registrar e Controlar Suas Despesas

Para que seu Dashboard Financeiro seja preciso, é essencial registrar todos os custos da academia. Este módulo ajuda você a não perder o controle dos gastos.

Passo a Passo para Adicionar uma Nova Despesa

1
No menu, acesse Despesas e clique em Nova Despesa.
2
Preencha o formulário com os detalhes do gasto (Descrição, Valor, Data).
3
Defina o Tipo: Classificar a despesa como Fixa (ex: aluguel, salários) ou Variável (ex: reparos, marketing) é crucial para uma análise financeira mais aprofundada.
4
Altere o Status: Inicialmente, a despesa pode ser cadastrada como Pendente. Assim que o pagamento for realizado, edite-a e mude o status para Pago. Isso evita pagamentos duplicados e mantém o fluxo de caixa correto.
5
Clique em Salvar.

Como Analisar a Saúde Financeira da Sua Academia

O Dashboard traduz todos os dados de pagamentos, vendas e despesas em gráficos e indicadores fáceis de entender.

Entendendo os Indicadores Principais

  • Receitas Totais: A soma de tudo que entrou no caixa (mensalidades + vendas).
  • Despesas Totais: A soma de todos os gastos que você cadastrou.
  • Lucro Líquido: O indicador mais importante (Receitas - Despesas). Mostra o que realmente sobrou.
  • Margem de Lucro: A porcentagem do seu faturamento que se converteu em lucro.
Ação Inteligente: Se o dashboard mostrar um alerta de pagamento pendente, vá ao módulo de Pagamentos para tomar uma ação. Se a margem de lucro cair, analise suas despesas para identificar possíveis cortes.

Soluções e Boas Práticas

Encontrou um problema ou quer dicas para usar o sistema de forma mais eficiente? Esta seção é para você.

Soluções: Verifique a ortografia, tente buscar apenas pelo primeiro nome ou confirme se o cadastro realmente existe.

Rotina Semanal: Verifique pagamentos pendentes, atualize o estoque e revise as despesas cadastradas.
Rotina Mensal: Analise o Dashboard Financeiro, verifique clientes sem fichas ativas e faça backup dos dados.

Suporte e Contato: Para suporte técnico, entre em contato através dos canais oficiais.
